# RakshaNetra - Animal Detection System

## Overview
RakshaNetra is an AI-based animal detection system that uses computer vision and deep learning techniques to detect animals from images and videos. The system uses YOLOv5 for accurate object detection and OpenCV for image processing.

## Features
- Real-time animal detection
- Image and video-based detection
- Animal identification using YOLOv5
- Bounding box visualization
- Fast and accurate detection results

## Technologies Used
- Python
- YOLOv5
- OpenCV
- PyTorch
- Deep Learning
- Computer Vision

## Project Workflow
1. User provides input image/video
2. Image frames are processed using OpenCV
3. YOLOv5 model detects animals
4. Bounding boxes are generated around detected animals
5. Detection results are displayed and saved

## Applications
- Wildlife monitoring
- Roadside animal detection
- Forest surveillance
- Human-animal conflict prevention
